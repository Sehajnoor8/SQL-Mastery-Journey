# **Notes**



1. SELECT: Used to retrieve data from a table.
**Example: SELECT emp_name FROM employees;**

2. DISTINCT: Used to remove duplicate values.
**Example: SELECT DISTINCT department FROM employees;**

3. WHERE: Used to filter rows based on conditions.

&#x20;  **Example: SELECT * FROM employees WHERE department = 'IT';**



**Key Points**

* SELECT * returns all columns.
* DISTINCT removes duplicates.
* WHERE filters data.
* SQL keywords are case-insensitive.
